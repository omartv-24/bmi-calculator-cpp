# BMI Calculator (C++)

This is a simple C++ program that calculates Body Mass Index (BMI) using user input.

## How it works
The user enters:
- Weight in kilograms (kg)
- Height in meters (m)

The program calculates BMI using the formula:
BMI = weight / (height * height)

## Output
The program displays:
- The BMI value
- The weight category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
